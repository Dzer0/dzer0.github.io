响应时间

在遇到网络问题的时候，我们一般会先通过 ping 这个工具来了解基本的情况。
httping 与 ping 类似，不过它不是发送 ICMP 请求，而是发送 HTTP 请求。利用 httping，我们可以测量出 Web 服务器跟网络的延迟。
